If you want to run the web application yourself, follow these steps:
1. Goto https://nodejs.org/en/ and download the LTS version of NodeJS and install it
2. Navigate to wintan_hospital directory using a command line, then type "npm install"
3. Then type "npm start"
4. Open your web browser and goto localhost:3000 or 127.0.0.1:3000
